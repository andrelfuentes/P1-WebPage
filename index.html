History of Lego

...

The Exploring the History of Lego project was done in my Data Science class. It was done through DataCamp and it was made so that through a sequence of structured tasks, you could explore Lego datasets related to colors, sets, and themes. I analyzed a database that included data on every LEGO set that has ever been sold; the names of the sets, what bricks they contain, what color the bricks are, etc. Overall, in this project I analyzed a dataset on every single Lego block that has ever been built.

Github link: Exploring-the-History-of-Lego

Page Replacement Algorithms

...

This project was a part of my Operating Systems class. The project was a three part code that implemented several page replacement algorithms in python. It reads a text file with written instructions and page numbers separated by spaces and implements the respective algortinms. For this project, the page replacement algorithms to work with are the FIFO page replacement algorithm, the Optimal page replacement algorithm, and the Working-Set Clock page replacement algorithm.

Github link: Page-Replacement-Algorithms
